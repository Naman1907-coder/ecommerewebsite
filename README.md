# 🛍️ E-Commerce Website UI (Tailwind CSS)

A modern, responsive e-commerce frontend built using **HTML** and **Tailwind CSS**. This project showcases a clean UI design including a navbar, product grid, category sections, and footer.

---

## 🚀 Features

* ✅ Fully responsive design
* ✅ Modern UI with Tailwind CSS
* ✅ Header with navigation & dropdowns
* ✅ Product listing grid
* ✅ Category sections (Men/Women)
* ✅ Mobile-friendly menu
* ✅ "Customers also purchased" section
* ✅ Hero section for promotions
* ✅ Newsletter + Footer section

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS (via CDN)
* Tailwind UI components

## ⚙️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

3. Double-click `index.html` or open it with Live Server

---

## 🌐 CDN Used

Tailwind CSS is loaded via CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/@tailwindcss/browser@4"></script>
```
